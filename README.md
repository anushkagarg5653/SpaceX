## Getting Started
* Clone the repo
   ```sh
   git clone https://github.com/anushkagarg5653/SpaceX.git
   ```
* Install NPM packages
   ```sh
   npm install
   ```
 * Run on
    ```sh
     http://localhost:3000/
    ```
